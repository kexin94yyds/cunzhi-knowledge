# 播客和公众号文章工作流

将语音录入内容转化为播客稿和公众号文章的完整工作流。

## 工作流概览

```
语音录入 → 文字 → 结构化播客稿 → 公众号文章
```

---

## 第一步：语音录入与初步结构化

1. 将语音录入并转成文字
2. 将文字发给 Opus 4.5，要求结构化

### 提示词模板

```
{内容}

请把以上的文字结构化一下，我要做成播客讲给大家。
```

---

## 第二步：应对内容过短的问题

如果初步结构化的内容太短，不足以支撑一篇完整的文稿，需要进一步扩展。

### 提示词模板

```
全面且结构化，太短了啊
```

完成后可直接制作成播客或视频发布。

---

## 第三步：从播客到公众号文章的转化

1. 使用 Typeless 将语音转文字的原始内容进行深度结构化

### 提示词模板（Typeless）

```
请将这段文字结构化一下
```

2. 将结构化后的内容复制，发给 Opus 4.5
3. 要求写成公众号文章

### 提示词模板（Opus 4.5）

```
{内容}

结合这个再写成一篇公众号文章
```

4. 如果内容偏短，让 AI "继续" 生成，通过多次交互形成完整文章

---

## 工具说明

| 工具 | 用途 |
|------|------|
| Typeless | 语音转文字 + 深度结构化 |
| Opus 4.5 | 内容结构化 + 文章生成 |
