# cunzhi-knowledge 全局知识库

## 用途

这是 iterate/cunzhi 系统的全局知识库，所有项目共享。

## 目录结构

- `problems.md` - 已解决的问题记录
- `patterns.md` - 可复用的解决模式
- `regressions.md` - 回归检查清单
- `conversations/` - 对话历史记录
- `prompts/skills/` - Skill 技能库
- `rules/` - 全局规则文件

## 与项目级 .cunzhi-memory 的区别

| 位置 | 范围 | 用途 |
|------|------|------|
| `项目/.cunzhi-memory/` | 项目级 | 当前项目的上下文 |
| `.cunzhi-knowledge/` | 全局级 | 所有项目共享的知识 |
